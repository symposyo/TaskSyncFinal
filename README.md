# TaskSyncFinal
Aplicación Android de gestión de tareas que demuestra el uso de arquitectura moderna , persistencia local y sincronización

### Stack Tecnológico
- **Lenguaje:** Kotlin + Corrutinas para asincronía.
- **UI:** ViewBinding y  Material Desing 3.
- **Base de datos:** Room (SQLite) para persistencia offline-first
- **Red:** Retrofit + OkHttp para consumo de APIs REST
- **Seguridad:** Manejo de Tokens con SahredPreferences
